Get Started

Up and running in seconds.

Terminal 
# Install
curl -sSL https://edge.hanzo.ai/install.sh | sh

# Or via cargo
cargo install hanzo-edge

# Run a model
hanzo-edge run zenlm/zen4-mini-gguf -p "Hello, world!"

# Start OpenAI-compatible server
hanzo-edge serve zenlm/zen4-mini-gguf --port 8080
Architecture

Three crates. One runtime.

hanzo-edge-core

Library

Core inference runtime. GGUF model loading, token sampling, tokenization, and hardware-accelerated tensor operations via Hanzo ML.

hanzo-edge

CLI Binary

Command-line interface. Run interactive sessions, inspect model metadata, benchmark throughput, and serve an OpenAI-compatible HTTP API.

hanzo-edge-wasm

WebAssembly

Browser inference via WebAssembly. Load models from URLs or IndexedDB and run completions entirely client-side.

API

OpenAI-compatible. Drop-in replacement.

Start a local server and use any OpenAI-compatible client library. No code changes needed.

HTTP Request 
curl http://localhost:8080/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{
    "model": "zen4-mini",
    "messages": [
      {"role": "user", "content": "Hi"}
    ]
  }'
Python 
from openai import OpenAI

client = OpenAI(
    base_url="http://localhost:8080/v1",
    api_key="not-needed"
)

response = client.chat.completions.create(
    model="zen4-mini",
    messages=[{"role": "user", "content": "Hello"}]
)

print(response.choices[0].message.content)
